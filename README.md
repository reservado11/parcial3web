Este proyecto es una tienda de maquillaje online. La idea es que cualquier persona pueda entrar, mirar productos, agregarlos al carrito y hacer pedidos. También hay una parte para que los administradores puedan manejar el inventario, categorías y revisar las órdenes. El objetivo es que sea una aplicación full-stack: Frontend en Vue 3 para que los usuarios tengan una interfaz bonita y fácil de usar. Backend en Express + TypeScript para manejar la lógica, las rutas y la conexión con los datos. Swagger para documentar la API y poder probar los endpoints sin complicarse. En resumen: una aplicación sencilla pero completa, que simula cómo funcionaría una tienda real de maquillaje.
